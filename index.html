Dependency Graph Directory

The following graphs were made using python to analyze public Swift repos containing package.swift files. The graph format was created using networkx and visualized and cleaned up using Gephi, and finally rendered using Sigma.js. You can click into into of the tabs above to see different dependency graphs.
